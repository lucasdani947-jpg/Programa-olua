Oque é if else e else? oque isso faz?

oque são "if, else e elseif"?

if else e else if são condições para variaveis!, mas como assim? tipo, if serve para condições com variavel, tipo pra fazer algo se a condição for verdadeira, else serve pra quando nenhuma das condições e verdadeira, fazer algo, elseif serve para adicionar mais de um if no mesmo codigo. exemplo: local "sua variavel" = "seu valor da variavel" if "sua variavel" == "seu valor dentro ou fora de aspas dependendo da forma usada" depois do valor do valor da variavel coloque um "then" sempre coloque ele no final da linha quando usar if. elseif "sua variavel" == "seu valor dentro ou fora de aspas" coloque "then" else "else serve pra quando nenhuma das condições for verdadeira" se quiser printar algo quando a condição for verdadeira ou não coloque um print debaixo da linha dos if ou else OBRIGAÇÃO: quando terminar um codigo com if sempre coloque "end" no final exemplo: "os seus ifs e prints" end.

Oque é while?

while é tipo loop, se vocÊ quiser fazer uma contagem de numeros veja o exemplo
exemplo:

local numero = 1 while numero <= 3 do -- da um loop se o numero for menor ou igual a 3 print("o numero é "..numero) -- printa "o numero é", o "..numero" serve pra colocar o valor da variavel, se você quiser escrever outra coisa depois do ..numero coloque "..numero.." e depois um " seu texto" numero = numero + 1 -- adiciona + 1 ao valor do numero end -- encerra o loop
